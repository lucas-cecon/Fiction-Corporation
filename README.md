# 🎬 Fiction Corporation

A web catalog focusing on science fiction movies and studios, developed as part of a technical course at SENAI. This project represents my first significant step into full-stack web development.

> ⚠️ **Historical Context (2023 Project)**
> *I am uploading this repository now just to archive it, but this project was actually developed back in 2023 during my early days at SENAI. It is not perfect—we were still learning the ropes, making mistakes, and discovering how things worked! I also didn't build this alone; I had the partnership of great classmates who are no longer working in the tech field. Please look at this code as a milestone of where my journey began, not as a reflection of my current technical level.*

## 📝 Project Overview
The goal was to create a functional platform where users could browse information about major sci-fi films and studios. Despite being a beginner project, it challenged the team to implement authentication and database integration for the first time.

## 🚀 My Role
- **Scrum Master:** Led the team throughout all development phases.
- **Lead Front-end Developer:** Responsible for the majority of the UI, including full responsiveness.
- **Scripting:** Developed most of the logic and scripts, including assistance with the login system.

## 🛠️ Tech Stack
- **Languages:** HTML, CSS, JavaScript, PHP
- **Database:** MySQL
- **Design:** Figma
